# 聚会桌游，谁是卧底

一个基于微信小程序的聚会桌游"谁是卧底"项目。

## 📁 项目结构

```
mp-undercover/
├── backend/              # 后端代码（Python Flask）
├── frontend/            # 前端代码（微信小程序）
└── README.md            # 项目说明
```

## 🚀 快速开始

### 后端

```bash
cd backend
./scripts/dev-start.sh
```

### 前端

在微信开发者工具中打开 `frontend/` 目录。
